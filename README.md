# VSCode Funny (KunKun Animation)

VSCode Funny is a lightweight extension that adds a playful KunKun animation effect while you type in the editor.

## Features

- **Typing Animation**: Displays an animated KunKun effect when text changes in the active editor.
- **Configurable Animation Size**: Lets you adjust the animation size to fit your preferred visual scale.
- **Configurable Right Offset**: Lets you control how far the animation appears from the right edge.
- **On/Off Switch**: Allows you to quickly enable or disable the typing animation.

This extension is designed to make everyday coding feel more fun and lively.
